# Discord Commands

This directory contains the Discord command specification for the Distribution interaction layer.

## Purpose

Document the available Discord commands, their inputs, outputs, and workflow.

## Structure

- `commands.md` — the main command spec with command descriptions and workflows.

## How to use

- Add new Discord command definitions to `commands.md`.
- Keep each command section focused on purpose, input, output, and workflow.
- Do not include implementation details or business logic in this spec file.

## Notes

- This folder is a documentation-only area for command design.
- The actual code implementation should live in the corresponding Discord handler directory.
