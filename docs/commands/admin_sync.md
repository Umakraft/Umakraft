# /admin_sync

Force an immediate data sync for all configured circles from uma.moe.

## Permissions
> 🔒 Requires **Manage Guild**

## Options
None.

## Behavior
- Reply is ephemeral (only visible to the user who ran it).
- Bypasses the normal hourly schedule and triggers a sync right away.
- Posts a result showing active member count, new members detected, and members who have left.
- Useful after circle roster changes or to verify data is up to date.

## Example
```
/admin_sync
```
