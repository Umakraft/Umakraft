# Command Reference

All 26 slash commands for the UmaKraft Circle Bot. Click any command to view its full documentation.

---

## 👤 Member Commands

Available to all members.

| Command | Description |
|---------|-------------|
| [/fan_gain](fan_gain.md) | Daily, weekly, and monthly fan gain + daily ranking |
| [/leaderboard](leaderboard.md) | Circle fan-gain leaderboard — daily / weekly / monthly |
| [/total_fan](total_fan.md) | Lifetime total fan count and circle rank |
| [/total_circlefan_gain](total_circlefan_gain.md) | Total accumulated fan gain for the entire circle |
| [/circle_master](circle_master.md) | Day-by-day Top 3 contributors for the current month |
| [/intercircleleaderboard](intercircleleaderboard.md) | Unified cross-circle fan-gain rankings |
| [/profile](profile.md) | Full profile dashboard — gains, records, monthly history |
| [/joindate](joindate.md) | Show when you joined the circle |
| [/memberlist](memberlist.md) | Full roster — active members + former members |
| [/search_trainer](search_trainer.md) | Search the trainer card database by name, rank, or skills |
| [/store](store.md) | Save your trainer card to the database |
| [/keep](keep.md) | Mark your trainer card as permanently kept |
| [/set_timezone](set_timezone.md) | Set your personal timezone for greeting messages |
| [/status](status.md) | Live bot health, sync status, and uptime |
| [/circle_status](circle_status.md) | Live sync status for all configured circles |
| [/help](help.md) | List all bot commands |

---

## 🔒 Admin Commands

Require **Manage Guild** or **Administrator** permission.

| Command | Permission | Description |
|---------|------------|-------------|
| [/link](link.md) | Manage Guild | Link a Discord account to an Uma.moe trainer |
| [/unlink](unlink.md) | Manage Guild | Remove a Discord ↔ Uma.moe link |
| [/link_list](link_list.md) | Manage Guild | Paginated list of all linked members |
| [/set_fans](set_fans.md) | Manage Guild | Set fan quota targets — daily / weekly / monthly |
| [/set_timezone](set_timezone.md) | — | *(member command — no admin restriction)* |
| [/admin_sync](admin_sync.md) | Manage Guild | Force an immediate data sync from uma.moe |
| [/admin_setjoindate](admin_setjoindate.md) | Manage Guild | Manually override a member's join date |
| [/admin_syncCards](admin_syncCards.md) | Administrator | Trigger a support card image sync from GameTora |
| [/test_milestone](test_milestone.md) | Manage Guild | Preview a milestone announcement without posting it |
| [/timeline_setup](timeline_setup.md) | Manage Guild | Configure the event timeline channel |
| [/timeline_post](timeline_post.md) | Manage Guild | Manually trigger a timeline post |
| [/warningsettings](warningsettings.md) | Administrator | View or update the warning engine configuration |
