# RECOMMENDED ACTIONS

Prioritized cleanup tasks:

1. Remove confirmed unused files
2. Merge duplicate logic modules
3. Lazy-load heavy services (Playwright, scrapers)
4. Simplify database access layers
5. Remove legacy code paths
6. Run dependency audit (npm + Knip)

---

# HOW THIS REPORT IS GENERATED

This report should ideally be generated using:

* AST analysis (Knip or similar tool)
* import graph scanning
* runtime usage observation
* manual review of feature usage

---

# NOTES

* Do NOT remove files without verifying runtime usage (cron jobs, event handlers, dynamic imports).
* Discord bots often appear "unused" but are event-driven.
* Scrapers and scheduled tasks may not be directly imported but still active.
