# `/profile` Command Specification

## Overview

`/profile` displays a member's permanent achievements, milestones, statistics, and progression history.

Unlike `/fangain`, which focuses on current progress, `/profile` highlights long-term accomplishments and serves as the member's permanent identity within the guild.

All responses are generated as an Image Report.

---

# Command

```text
/profile
```

Default behavior displays the requesting member.

Future enhancement

```text
/profile @member
```

Allows viewing another linked member's profile.

# Design Principles

* Permanent member identity.
* Focus on long-term accomplishments.
* Celebrate progression rather than only current performance.
* Reward consistency with achievements and titles.
* Keep all statistics data-driven.
* Every profile is generated as an Image Report.

---

# Milestone Goal

The `/profile` command should become the member's permanent showcase.

A single image should instantly communicate:

* Who the member is.
* What they have accomplished.
* Their highest milestone.
* Their achievements.
* Their streaks.
* Their personal records.
* Their current status.
* Their progression toward the next milestone.

The profile should evolve naturally as the member continues playing, making it a living record of their journey within the circle.
